#0x01-emacs
This project is about learning the emacs tool, with all its commands and shortcuts

# Task's Files
1. **0-opening** : The command to open a file within emacs
2. **1-saving** : The command to save changes of the current file
3. **2-cutting** : The commad to cut *kill* the current line
4. **3-pasting** : The command to past something
5. **4-searching** : The command to search forward
6. **5-undoing** : The command undo
7. **6-quitting** : The command to quit emacs
8. **100-tetris** : The command to play tetris inside emacs
9. **101-doctor** : The command to talk to my doctor
