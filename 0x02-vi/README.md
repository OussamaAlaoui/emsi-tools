#0x02-vi
This project is about learning vi tools and commands.

# Tasks 
1. **0-inserting** :  The command to insert text before the cursor
2. **1-cutting** : The command to delete and cut the current line
3. **2-pasting** : The command to paste the lines in the buffer into the text after the current line.
4. **3-undoing** : The command to undo what i just did
5. **4-exiting** : The command to quit Vi
6. **5-beginning_line** : The command to move the cursor to the start of the current line
7. **6-end_line** : The command to move the cursor the end of the current line 
8. **100-move_to_line** : The command to move the cursor to the line 32
9. **101-delete_line** : The command to remove the current and the previous line at the same time
