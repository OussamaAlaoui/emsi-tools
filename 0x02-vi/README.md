#0x02-vi
This project is about learning vi tools and commands.

# Tasks 
1. **0-inserting** :  The command to insert text before the cursor
2. **1-cutting** : The command to delete and cut the current line
3. **2-pasting** : The command to paste the lines in the buffer into the text after the current line.
4. **3-undoing** : The command to undo what i just did
